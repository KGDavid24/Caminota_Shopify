# Vopseaua BioCalce
Kerakoll Biocalce® Tinteggio, pastă de var naturală, este o zugrăveală murală naturală eco compatibilă de hume colorate naturale şi pastă selecţionată din var pur. Este ideală în Green Building şi în Restaurarea Istorică.
Compoziție: var, ulei de pin
Beneficii: sprijină sănătatea mediului interior, un bacteriostatic şi fungistatic natural, permite trecerea vaporilor, nu permite trecerea apei lichide
